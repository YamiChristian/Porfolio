McDonald's Android App Redesign

Web Design
Project Overview
The objective of the project was to create a user friendly rebrand and rebuild of the Android app for McDonald's Restaurants.
My Contributions
I was in charge of the project from start to finish. From conducting user research and compiling data. To creating and finalizing the prototypes of the application.
With how the modern world has became more dependent on applications to order food for delivery or pick up. Good UX/UI has never been more important. My job was to develop a working prototype that would enhance the strengths of the current application as well as improving upon the weak parts such as the lack of images and the uninteresting menus.

Gathering Data
My first act on this project was to gather information. I gathered information by doing the following:
  • Conduct user surveys
  • Researched Android application and website
  • Used Personal experience using the services and applications for an expended period of time.

Personas

After gathering all of my data from my user interviews as well as my survey, I was able to organize my data into a few different sections including; pleasant points, pain points, items that help with retention, themes, and lastly key insights.

Problem

Users need an easy to navigate app that gives gives them an understanding of what the product they want looks like without having to deal with reading line after line of text. The current application also does not give you access to the full menu right away and restricts access based on time of day. it also spreads out information through multiple pages.

Target User Base

People who regularly use the McDonald's App.
Those who want a quick yet efficient experience with McDonald's.

Minimum Viable Product

An redesigned application giving users the same great service that user's have come to expect from McDonald's but with a better. The application will include the following core features:

  • Visually appealing enhanced graphics that put visuals in the forefront.
  • New layout to help users easily navigate the app.
  • Combining pages that have relevant information.

Solution

An updated experience that lets users order their favorite meals with ease with larger images and less pages to navigate in order to find information.
